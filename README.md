## To submit requests for additional cells:

1. At the top of the repository, click on `issues`
2. Create a new issue
3. Title the issue: **Bingo Cell Idea**
4. Put yout idea in the description
5. Sumbit

If *the council* deems the idea good enough, it will be added to the list

**Please look over the list of current bingo cells located in `src/list.db` before submitting**

### Contributing

If you have experience in web development, then you probably have way more experience than I do. Feel free to correct any issues to make the website better! ALl I ask is that you don't add any new features that go outside the website's intended purpose - playing bingo.
